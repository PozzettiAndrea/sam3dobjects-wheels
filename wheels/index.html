SAM3DObjects Wheels

Pre-built CUDA extension wheels for ComfyUI-SAM3DObjects.

RTX 50 Series (Blackwell) Support: Use +cu128 wheels for RTX 5070/5080/5090.

Packages (PEP 503)

gsplat
nvdiffrast

Quick Install

# For RTX 50 series (CUDA 12.8 + PyTorch 2.8.x):
pip install gsplat==1.5.0+cu128 nvdiffrast==0.3.3+cu128 \
  --extra-index-url https://pozzettiandrea.github.io/sam3dobjects-wheels/

# For RTX 20/30/40 series (CUDA 12.4 + PyTorch 2.4.x):
pip install gsplat==1.5.0+cu124 nvdiffrast==0.3.3+cu124 \
  --extra-index-url https://pozzettiandrea.github.io/sam3dobjects-wheels/
