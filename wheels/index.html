SAM3DObjects Wheels

Pre-built CUDA extension wheels for ComfyUI-SAM3DObjects.

RTX 50 Series (Blackwell) Support: These wheels include sm_120 architecture support for RTX 5070/5080/5090.

Available CUDA Versions

CUDA 12.8

For PyTorch 2.8.x - Required for RTX 50 series (Blackwell)

Also works with RTX 20/30/40 series

pip install gsplat nvdiffrast --find-links https://pozzettiandrea.github.io/sam3dobjects-wheels/cu128/

Packages

  • gsplat - Gaussian Splatting rasterization (source)
  • nvdiffrast - Differentiable rendering (source)
